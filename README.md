# vaccine-cold-chain

**EN**

Research and undergraduate thesis project (UFSC, 2022). Development of a temperature monitoring and forecasting system for vaccine cold chains using artificial intelligence.

**Repository structure**

- `forecast/` — temperature prediction model (Python, machine learning)
- `android-app/` — Android application for real-time temperature monitoring (Java)
- `sheets-integration/` — Google Sheets backend integration (Java)

**Technologies**

`Python` `Java` `Machine Learning` `Android`

---

**PT**

Pesquisa e Trabalho de Conclusão de Curso (UFSC, 2022). Desenvolvimento de um sistema de monitoramento e previsão de temperatura para redes de frios de vacinas utilizando inteligência artificial.

**Estrutura do repositório**

- `forecast/` — modelo de previsão de temperatura (Python, aprendizado de máquina)
- `android-app/` — aplicativo Android para monitoramento de temperatura em tempo real (Java)
- `sheets-integration/` — integração com Google Sheets como backend de dados (Java)

**Tecnologias**

`Python` `Java` `Machine Learning` `Android`
