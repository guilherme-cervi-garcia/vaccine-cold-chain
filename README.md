# vaccine-cold-chain

**EN**

Undergraduate thesis project (UFSC, 2022). Development of a temperature monitoring and forecasting system for vaccine cold chains using artificial intelligence.

**Repository structure**

- `forecast/` — temperature prediction model (Python, machine learning)
- `monitor-android/` — Android application for real-time temperature monitoring (Java)

**Technologies**

`Python` `Java` `Machine Learning` `Android`

---

**PT**

Trabalho de Conclusão de Curso (UFSC, 2022). Desenvolvimento de um sistema de monitoramento e previsão de temperatura para redes de frios de vacinas utilizando inteligência artificial.

**Estrutura do repositório**

- `forecast/` — modelo de previsão de temperatura (Python, aprendizado de máquina)
- `monitor-android/` — aplicativo Android para monitoramento de temperatura em tempo real (Java)

**Tecnologias**

`Python` `Java` `Machine Learning` `Android`
